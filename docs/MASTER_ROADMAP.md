# Master Roadmap v2.0 (English)

## Long-Term Mission

Build trusted AI systems that help people make better decisions.

Project Origin is the first product.

ReconOS is the second product.

Origin Engine is the reusable technology extracted only after successful product validation.

---

# Core Strategy

**Product First. Engine Later.**

We do not build a platform first.

We build a valuable product.

When reusable patterns naturally emerge, we extract them into the Origin Engine.

---

# Phase 1 — Foundation ✅

Purpose

Establish immutable company principles.

Completed

* README
* PRD
* Mission
* Brand DNA
* Vision
* Core Values

These documents form the company's constitution.

---

# Phase 2 — MVP

Goal

Deliver a working product as quickly as possible.

Target

A founder completes an interview and receives a professional Brand Strategy Report.

Workflow

Founder Interview

↓

Knowledge Extraction

↓

Reasoning

↓

Brand Generation

↓

Evaluation

↓

Final Report (PDF)

No unnecessary features.

Only one complete workflow.

---

# Phase 3 — Validation

Goal

Acquire paying users.

Target

10 paying customers.

Objectives

* Validate report quality
* Improve reasoning
* Measure LLM costs
* Optimize workflow

Everything else is secondary.

---

# Phase 4 — Product Evolution

Improve

* Brand quality
* Evaluation accuracy
* Report generation
* User experience

Expand carefully based on user feedback.

---

# Phase 5 — Origin Engine Extraction

Only after repeated validation.

Extract reusable components such as:

* Knowledge Layer
* Reasoning Layer
* Evaluation Layer
* Report Layer
* LLM Provider Layer

Do not extract code prematurely.

---

# Phase 6 — ReconOS

Use lessons learned from Project Origin.

Apply the reusable components where appropriate.

Do not assume complete code reuse.

Reuse proven patterns instead.

---
