# Project Origin Memory
Version: Alpha 0.1

---

# Mission

Project Origin is NOT a brand name generator.

Project Origin is an AI-powered Brand Intelligence Operating System.

Its purpose is to help founders make better branding decisions through structured reasoning.

---

# Vision

Build the world's most intelligent AI Brand Strategy Platform.

The platform should eventually support:

- Company Naming
- Product Naming
- Service Naming
- AI Agent Naming
- Character Naming
- Brand Positioning
- Brand Identity
- Brand Architecture
- Brand Strategy

Naming is only one component.

---

# Core Philosophy

Meaning comes before names.

A good brand name is the result of strategic thinking,
not random creativity.

Project Origin should never generate names first.

It should first understand:

Founder

↓

Business

↓

Strategy

↓

Meaning

↓

Language

↓

Name

---

# Fundamental Principles

## 1.

LLMs are evaluators.

Not generators.

Project Origin owns the generation engine.

GPT evaluates.

Project Origin creates.

---

## 2.

Every conclusion must be explainable.

No black-box outputs.

Every recommendation should have reasoning.

---

## 3.

Architecture over Prompt Engineering.

The system should improve by improving engines,
not by writing longer prompts.

---

## 4.

Each engine has one responsibility.

No engine should perform multiple unrelated tasks.

---

## 5.

Everything should be testable.

Every core engine must have automated tests.
