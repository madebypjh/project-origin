# Brand Strategy Frameworks

## Purpose

This document defines the strategic branding frameworks used by Project Origin.

These frameworks serve as the reasoning foundation for the Knowledge Builder and Prompt Builder.

Project Origin does not generate names randomly.

Instead, it follows structured strategic thinking used by professional branding consultants.

---

# Framework 1 — Golden Circle (Simon Sinek)

Purpose:
Identify the company's core purpose before discussing products.

Questions:

* WHY does this company exist?
* HOW does it create value?
* WHAT does it actually build?

Used in:

* Knowledge Builder
* Founder Analysis
* Mission Statement

---

# Framework 2 — Brand DNA

Purpose:

Extract the immutable identity of the company.

Components:

* Purpose
* Vision
* Core Values
* Personality
* Differentiation

Used in:

* Brand Identity
* Positioning
* Narrative

---

# Framework 3 — Brand Positioning

Purpose:

Determine where the brand should exist in the customer's mind.

Components:

* Target Audience
* Competitive Alternative
* Unique Value
* Reason to Believe

Output:

Positioning Statement

---

# Framework 4 — Brand Archetypes

Purpose:

Define the emotional identity of the brand.

Examples:

* Creator
* Sage
* Hero
* Explorer
* Caregiver
* Magician
* Rebel
* Everyman
* Innocent

Used in:

* Brand Personality
* Tone of Voice
* Narrative

---

# Framework 5 — Value Proposition Canvas

Purpose:

Understand customer jobs, pains, and gains.

Output:

* Customer Needs
* Product Value
* Value Alignment

---

# Framework 6 — Naming Principles

Project Origin evaluates every candidate name using the following dimensions:

* Meaning
* Memorability
* Pronunciation
* Domain Availability (future)
* Trademark Potential (future)
* Scalability
* Emotional Impact

---

# Framework 7 — Decision Intelligence

Project Origin does not optimize for creativity alone.

Every recommendation should maximize:

* Strategic Alignment
* Long-term Scalability
* Brand Consistency
* Founder Intent
* Market Differentiation

This framework is the core reasoning philosophy of Project Origin.
