# Project Origin

> Reveal What Matters.

Project Origin is an AI-powered Brand Intelligence Platform.

It is designed to help founders, builders, and teams discover brand names that align with their mission, values, and long-term vision.

Rather than generating random names, Project Origin combines structured reasoning, scoring, filtering, and AI-assisted evaluation to reveal the names that truly matter.

## Mission

Reveal What Matters.

## Why Project Origin Exists

A brand is not just a name.

A brand is a promise.

Project Origin exists to help people discover names that are not only memorable, but meaningful, usable, and aligned with a clear purpose.
